User Stories:

Story #1:
As a User 
I can list all the ingredients I have at home.
So that I can see a list with all my ingredients and their exact quantity.
-----------------------------------------------------------------------------
story #2:
As a User
I can add more of the same ingredient to my ingredient list.
So that I can see in the list how much of the same ingredient I have altogether and if buy more of the same kind of ingredient only the new quantity will be added to the list.
.....................
Acceptance criteria:

Given there is 1 kg of flour in my list
When I add 0.5 kg more flour to the list and submit
Then the new quantity will be shown in the list next to the ingredient name "flour".
